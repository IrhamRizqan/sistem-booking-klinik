# UI Guidelines

This project uses Bootstrap 5.

Use Hallmark as the primary UI reference.

Follow Hallmark principles for:

- spacing
- typography
- layout
- component hierarchy
- interaction states
- microinteractions

Never imitate AI-generated dashboard styles.

Never use Bootstrap default appearance without customization.

Always use Bootstrap utility classes before writing custom CSS.

Prefer reusable partials.

Icons

Bootstrap Icons only.
